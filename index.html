Projects

Selected projects that show how I approach problems, work with data, build models, and evaluate results.

Sleep Arousal Detection with Deep Learning

2026 – Present
Problem
This project aims to detect sleep arousal events from multi-channel physiological signals. The goal is to build a deep learning system that can identify important signal patterns and improve performance on a real biomedical classification task.
Data
The project uses multi-channel physiological signal recordings from the PhysioNet Computing in Cardiology 2018 dataset. The main inputs are time-series signals collected during sleep studies for sleep arousal detection.
Approach
I worked on data preprocessing, label handling, baseline model training, and experiment analysis. The project explores deep learning methods including CNN- and Transformer-based models for physiological signal classification.
Outcome / Impact
The baseline model achieved an AUPRC of 0.412. This project helped me build practical experience in signal preprocessing, model experimentation, evaluation, and technical documentation.
My Contribution
I contributed to preprocessing logic, model experimentation, result analysis, and project documentation. This project is currently in progress.

Flappy Bird Agent Training with Q-Learning

2026
Problem
This project aimed to build an AI agent that can learn to play Flappy Bird automatically. The objective was to apply reinforcement learning concepts to a game environment and improve performance through training.
Data
The project uses game-state information generated during gameplay to represent the environment and support the agent’s decision-making process.
Approach
I implemented a Q-learning agent with an epsilon-greedy exploration strategy in Python. I trained the agent across 20,000 episodes and improved performance through repeated testing and tuning.
Outcome / Impact
This project demonstrated how reinforcement learning can improve decision-making through repeated interaction with an environment. It also strengthened my understanding of experimentation, reward design, training analysis, and algorithm evaluation.
My Contribution
This was an individual course project, and I completed the implementation, training, evaluation, and result analysis by myself.
